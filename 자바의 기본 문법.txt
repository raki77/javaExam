자바의 기본 문법

1. 식별자

- 자바프로그래밍을 하는데 필요한 구성요소를 구별하는 이름
- 식별자 정의 규칙
  => 중간에 공백문자, 특수문자 사용하면 안됨.
 cf) _(언더바), $(달러)는 허용.

- 클래스명 : 첫 글자 대문자로 시작.
- 메소드명(함수) : 첫 글자는 소문자로 시작.

- 메소드 명이 두 글자 형태일때는 소문자 - 대문자(moveTest[])
 
* 상수(constant)명 정의규칙

  => 자료를 담을 수 있는 메모리 공간으로 프로그램이 끝날 때까지 자료가 변하지 않는 것.
  - 상수의 모든 문자는 대문자로 시작.
  - 하나 이상의 단어 연결시 _로 연결

* 변수(variable) 명의 정의규칙

  => 자료를 담을 수 있는 메로리 공간으로 객체의 색상이나 크기 같이
     변할 수 있는 상태를 나타낸 것. (메소드로 변수의 일종)

 

2. 예약어 (키워드)

- 미리 자바 언어에서 사용되는 메소드 이름이나 클래스명이 정의되어 있기
  때문에 프로그램에서 사용하지 않았으면 하는 이름들..
- 사용하게 되면 에러나 결과가 잘못 나옴.


3. 기본 자료형 (Data Type) - int 4byte형태

- 논리형 -> true/false boolean 1bit false (기본값)
  문자형 -> char 2byte 0부터 65,536 /u0000 (기본값, u는 유니코드, 2byte= 16bit, 00는 1byte=8bit로 표현, 0는 4bit )
 cf) 0042 -> 0000 , 0000 , 0100 , 0010 (아스키 코드값을 /16 나누어 2byte로 표현)
  정수형
 byte 1 byte
 short 2 byte
 int  4 byte -2,147,483,648부터 2,147,483,648
 long 8 byte

 byte -> short -> int -> long 묵시적 형변환
 long -> int -> short -> byte 명시적 형변환 (바꾸고자 할때 꼭 바꿀 형을 선언해야 한다.)

- 실수형 -> float 4 byte
     double 8 byte
